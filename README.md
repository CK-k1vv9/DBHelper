# DBHelper

## 简介

1. 支持Oracle、MSSQL、MySQL、SQLite四种数据库
2. 方便扩展以支持更多关系数据库
3. 有配套的Model生成器
4. insert、update、delete操作无需写SQL
5. 查询使用原生SQL
6. 查询结果通过映射转成实体类或实体类集合
7. 支持参数化查询，通过SqlString类提供非常方便的参数化查询
8. 支持连接多个数据源

## 优点

1. 代码实现比较简单，有经验的程序员可以自己修改、扩展源码
2. 查询使用原生SQL

## 缺点

1. 不支持Lambda表达式、链式调用等语法糖
